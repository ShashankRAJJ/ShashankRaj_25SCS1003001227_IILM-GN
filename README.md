# ShashankRaj_25SCS1003001227_IILM-GN
Automatic Vehicle Number Plate Recognition (ANPR) using AI/ML

This project implements an intelligent Automatic Vehicle Number Plate Recognition (ANPR) system using Python. It leverages computer vision and machine learning techniques to detect vehicle number plates from images or video streams and accurately extract the text.

The system uses a Cascade Classifier (Viola-Jones algorithm) for real-time number plate detection and Tesseract OCR for character recognition. Extracted data is processed and stored in an SQL database along with timestamps for efficient tracking and retrieval.

🚀 Key Features:
	•	Real-time number plate detection using OpenCV
	•	Accurate text extraction using Tesseract OCR
	•	Handles variations like lighting, angles, and motion
	•	Stores recognized data with timestamps in a database
	•	High detection accuracy (up to 99.8% for images)

🛠️ Tech Stack:
	•	Python
	•	OpenCV (Haar Cascade Classifier)
	•	PyTesseract (OCR)
	•	SQL Database

📊 Applications:
	•	Traffic monitoring systems
	•	Automated toll collection
	•	Parking management
	•	Vehicle tracking & security
	•	Law enforcement and surveillance
